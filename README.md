# spring-cloud-demo

1. Install Maven

2. Initialize Projects
   	 
> workDir=/home/xwp/git/spring-cloud-demo
> cd $workDir/spring-cloud-config-server
> mvn dependency:tree
> mvn eclipse:eclipse

3. Run the applications
> workDir=/home/xwp/git/spring-cloud-demo
> cd $workDir/spring-cloud-config-server
> mvn spring-boot:run

4. Open Eureka
   http://localhost:8763
